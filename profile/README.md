# SaisonManager 🏒
Inoffizielle Dokumentation für den Floorball-Deutschland SaisonManager [saisonmanager.de](https://saisonmanager.de)

Da der Saisonmanager derzeit keinerlei Dokumentation zu seiner Api hat, ist hier eine (hoffentlich) aktuelle Beschreibung der Schnittstelle.

## Was ist hier zu finden?

### [OpenApi](https://github.com/SaisonManager/OpenApi)
- OpenApi Dokumentation beschreibt die aktuelle Schnittstelle von [saisonmanager.de/api/v2](https://saisonmanager.de/api/v2)
- aus der OpenApi-Spezifikation [generierte Packages](https://github.com/SaisonManager/OpenApi/packages) für gängige Programmiersprachen & Frameworks  
    (NPM - JS/Typescript, nuget - C# .NET)
